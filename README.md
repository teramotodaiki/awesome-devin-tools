## Development Tools

Tools that enhance Devin's development capabilities:

- [@yuiseki/devin-gyazo](https://github.com/yuiseki/devin-gyazo) - A collection of automation scripts for capturing and uploading screenshots to Gyazo, enabling Devin to easily share visual information during development tasks.
  - Features browser screenshot capture with metadata support
  - Supports shell output capture with ANSI color
  - Integrates directly with Gyazo API

4. Check your spelling and grammar
